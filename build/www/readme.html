BonziCHAT

What the hell is this?

BonziCHAT is a node.js and socket.io based chat client featuring everyone's most hated loved purple gorilla. The client also uses speak.js to provide text-to-speech voices. Not exactly the original BonziBUDDY voice, but hey, it works.

Why did you make this? Why WOULD you make this?

I was bored. I'm sorry. Please spare me.

How do I use it?

  1. Enter a nickname (under 25 characters). If you don't enter one, you'll be named "Guest".
  2. Optionally, enter a room ID. If you want to join someone else, their room ID will be in the bottom-right corner of the screen.
    • If you enter a room ID that doesn't exist, you will be placed in a private room which can only be joined by people you share the ID with.
  3. Be a BonziBUDDY. There are 2 variants of him, It's either a parrot or the infamous gorilla.

What are the chat commands?

If you see any brackets, they indicate a placeholder. Don't type them in.

  • /joke - Tell a "joke".
    • /fact - Tell a already known fact.
      • /linux - I'd just like to interject for a moment.
        • /pawn - Hi, my name is BonziBUDDY.
          • /name [name] - Change your name.
            • There is a 25 character limit on names.
          • /speed [speed] - Change your voice's speed.
            • Max value is 275, min value is 125.
          • /pitch [pitch] - Change your voice's pitch.
            • Max value is 125, min value is 15.
          • /color [color] - Change your BonziBUDDY's character! The ones available are:
            • peedy
            • purple
            • pink
            • red
            • green
            • blue
            • black

          Chat rooms are for the most part unmoderated, If you are annoyed by somebody, Use the 'Mute' button. If shit starts getting spambotty, I might start using my banhammer.
